# 🎓 Education Data Dashboard

## 📌 Overview
This dashboard provides a detailed analysis of education data including student performance, course distribution, semester-wise trends, and pass/fail statistics. The dashboard is designed to be **interactive**, enabling easy exploration of data through slicers, charts, and KPI cards.

## 🛠 Tools Used
- Microsoft Power BI (for visualization and dashboard creation)
- Python – For data cleaning and analysis (read education.csv and processed into  clean_education.xlsx)
- Excel – Cleaned dataset (clean_education.xlsx) for analysis and dashboard input

## 🖥 Dashboard Features
- **KPI Cards:** Quick overview of key metrics like total students, average marks.
- **Slicers:** Filter by course, semester metrics for granular analysis
- **Interactive Charts:** Distribution, Attendence, Average Marks
- **Insights-driven Layout:** Designed to highlight the most important insights immediately

## 📊 Key Insights
1. **Course-wise Pass/Fail Analysis:**
   - MCA students have the **highest pass rate**
   - B.Tech students have the **highest fail rate**

2. **Attendance vs Marks:**
   - Attendance has **minimal impact on marks**
   - Students with **lowest attendance often have higher marks** compared to others

3. **Semester-wise Performance:**
   - **4th Semester** students have the **highest average marks**
   - **2nd Semester** students have the **lowest average marks**

4. **Overall Pass vs Fail Distribution:**
   - Pass: 292 students (**49.49%**)  
   - Fail: 298 students (**50.51%**)  
   - Overall, **pass and fail counts are almost equal**
